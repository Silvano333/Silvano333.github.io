GraphsJ 2.0

  • Launch GraphsJ 2 to run graph algorithms:
    Launch using the default theme Launch using the Nimbus theme
  • To develop your own, custom algorithms in Java, or just to have a look at the source code, download the GraphsJ 2 Software Development Kit (SDK)
  • Download the pseudocode for the standard algorithms implemented by this version of GraphsJ 2
  • Download some example files

Please, note: Java 1.6 or higher is required in order to run the application or to create your own algorithms.
