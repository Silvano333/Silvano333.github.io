Requirements

This is a Java application: the latest version of Java is therefore recommended.

Installation instructions

To install this app, you can:

  • Download its binary zip file, decompress it wherever you want and run one of the scripts in its bin folder

  • Alternatively, you can install it via MoonDeploy, just by clicking on Run with MoonDeploy and opening the app descriptor file with MoonDeploy.
