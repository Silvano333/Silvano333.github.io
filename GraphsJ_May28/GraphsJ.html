Please, note: Java 1.6 or higher is required in order to run the application or to create your own algorithms.



What's new in version 1.2

  • WeightQuantity class introduced, which is used to elegantly represent weights and capacities
  • StandardEdge interface introduced: edges implementing this interface can reference the currently running StandardAlgorithm (refer to the javadoc for further information)
  • Some bugs fixed in the implementation of CPM
  • New file format introduced: files created with the previous versions of the program won't be opened!
  • GraphObjectList, VertexSet, EdgeSet, VertexList and EdgeList classes introduced, to easily manage collections of vertexes and edges.
  • BasicGraphObject, BasicVertex and BasicEdge were renamed to GraphObject, Vertex and Edge respectively
  • Some methods removed or modified in GraphKeeper, Vertex and Edge, since they are now supported by the new classes
  • Fixed a visualization bug in the CommonDialog: on some desktop themes, the OK and Cancel button weren't shown appropriately
  • HTML help available to algorithms
  • Some small code enhancements.
